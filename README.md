# Basic Calculator App using Angular

## Project Overview

This project is a simple **Basic Calculator App** built using **Angular**. It demonstrates key concepts and practices in Angular such as **Component-based architecture**, **Two-way data binding**, **Template-driven forms**, **Bootstrap styling**, and **Control Statements**. The app allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Simple calculator interface
- Supports four basic arithmetic operations: addition, subtraction, multiplication, and division
- Utilizes Angular's **Two-way data binding** to reflect user input
- Styled using **Bootstrap** for responsive design
- Built with **Template-driven forms** for input handling

## Technologies Used

- **Angular**: Framework for building the application
- **HTML**: For structure and layout of the app
- **CSS**: For styling, along with **Bootstrap** for responsive design
- **TypeScript**: For logic implementation in Angular components
